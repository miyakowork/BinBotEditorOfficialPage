# BinBotEditor 官方网站

BinBotEditor 的中文产品官网。首版面向 macOS，当前下载状态为“即将发布”。

## 本地开发

```bash
npm install
npm run dev
```

默认开发地址为 `http://localhost:5173`。

## 验证

```bash
npm test -- --run
npm run build
```

生产构建输出到 `dist/`。网站为纯静态 React 应用，不依赖后端服务、远程图片或分析脚本。

## 当前范围

- 中文单页官网
- macOS 产品介绍
- 产品功能、内置格式化和本地优先说明
- 统一的“macOS 版即将发布”状态
- 桌面、平板和手机响应式布局

真实下载文件和发布日期将在 BinBotEditor 首次公开发布时补充。
